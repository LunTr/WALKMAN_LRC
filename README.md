# WALKMAN_LRC:解决索尼播放器歌词烦恼

如果你是尊贵的索尼WALKMAN用户，你可能会高兴地发现：

### 系统自带的播放器不能识别歌词文件！

这可能由于歌词(.lrc)文件的时间轴不正确。歌词文件以形如[00:00.000]的格式存储，而索尼的播放器只能识别[00:00.00]的时间轴并将其判定为歌词。

这个项目提供了指向特定目录、一键处理目录下所有lrc文件的功能，使你心爱的ZX707或金砖能畅享歌词。观看精美的歌词以系统默认字体渲染的小字号字幕滚动，真是令人无比偷税的体验。

如果lrc文件的文字Unicode编码不是UTF-8，那么需要将文件转换成UTF-8。

---

项目采用轻量的Tauri框架，后端使用golang驱动，简洁高效。

启动方式：进入项目文件夹打开终端，输入：

```
cd tauri-app
npm.cmd run dev
```

初始化编译可能需要一段时间，这是正常现象。

也可以直接双击lrc-proc.exe启动项目，或安装release，这个过程不需要编译耗时。

目前确认支持机型：本人的ZX505，欢迎大家积极使用提意见。
